## Indobase-status

Shows status of the Indobase local development stack.

Requires the local development stack to be started by running `Indobase start` or `Indobase db start`.

You can export the connection parameters for [initializing Indobase-js](https://Indobase.com/docs/reference/javascript/initializing) locally by specifying the `-o env` flag. Supported parameters include `JWT_SECRET`, `ANON_KEY`, and `SERVICE_ROLE_KEY`.
