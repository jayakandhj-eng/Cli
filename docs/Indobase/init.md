## Indobase-init

Initialize configurations for Indobase local development.

A `Indobase/config.toml` file is created in your current working directory. This configuration is specific to each local project.

> You may override the directory path by specifying the `Indobase_WORKDIR` environment variable or `--workdir` flag.

In addition to `config.toml`, the `Indobase` directory may also contain other Indobase objects, such as `migrations`, `functions`, `tests`, etc.
